Snaqe - Snake AI

This is the classic game Gluttonous Snake played by A-star algorithm and developed in Qt framework, how about calling it "Snaqe"?

New

The Snake has become a game ideology. My purpose is to let the computer replace people to pass the game. Before that, I had coded a Snake game first. Then I embedded the AI into it.

The path searching algorithm for our snake is the A-star algorithm, and it could work well when the snake length was not too long. However, as the snake length grew, it would generate so many holes on the game map. Filling these holes by the snake would take a lot of time. Therefore, I modified the AI decision: call the A-star algorithm when snake length is less than a threshold, elsewhere follow Hamilton path. It worked better but still caused the snake to trap into a death loop sometimes, so I am still working on it.

QPoisson - Photo Editor with Poisson Blending

The 'QPoisson' is not just a photo editor capable of ordinary transformations in image processing, such as mirroring, rotation, color inverse, turning into greyscale, etc., but a carrier for implementing Poisson blending algorithm. That is to cut a fragment, which is smaller than the aim picture, from the source picture, and then seamlessly blend it into the aim picture.

FluidS - Fluid Simulator

Its core algorithm of the simulation process refers to Robert Bridson's Fluid Simulation for Computer Graphics. It uses the numerical solution of Navier-Stokes equations to predict the density and velocity distribution of particles in each frame displayed in the OpenGL widget of Qt. There are two resolution options for users: 64x64 and 128x128. Users can also change display mode between density and velocity in 7 colors.
